// Tải và chạy script trên máy
1

Tải script về máy

Click nút bên dưới để tải file PowerShell về máy tính Windows của bạn.

audit_pc5.ps1
Script PowerShell
run_audit5.bat
File chạy nhanh (double-click)
2

Chạy script với quyền Admin

Nhấn phải chuột vào run_audit5.bat → Run as administrator. Hoặc mở PowerShell Admin và chạy:

powershell -ExecutionPolicy Bypass -File audit_pc5.ps1

Script sẽ hỏi tên và email (đã tự điền sẵn nếu bạn nhập ở bước 1).

3

Lấy file kết quả CSV

Sau khi chạy xong, file CSV được tạo tại:

%USERPROFILE%\Desktop\PC_Audit\audit_TENMAYTÍNH.csv
4

Upload file CSV lên web này

Quay lại trang web và upload file CSV để xem báo cáo đầy đủ.
